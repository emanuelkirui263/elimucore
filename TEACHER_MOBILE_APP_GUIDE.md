### "App keeps crashing"

```
1. Force close app (swipe up on iPhone, back button on Android)
2. Restart your phone
3. Try opening app again
4. If crashes again, delete and reinstall
5. Contact your school IT support
```

### "Can't login on mobile"

```
1. Check internet connection
2. Verify email is correct (same as web)

5. Restart phone and try again
6. Contact school administrator
```

```
1. Check internet connection (WiFi preferred)
2. Open app settings
3. Look for "Sync now" button
4. Wait 2-3 minutes
5. Check if marks appear
6. If still not synced, contact support
```

### "App is slow"

```
1. Close other apps (frees memory)
2. Clear app cache: Settings â†’ Apps â†’ ELIMUCORE â†’ Storage â†’ Clear Cache
3. Update to latest version
4. Restart phone
5. Use WiFi instead of mobile data
```

### "Notification not working"

```
1. Go to Settings â†’ Notifications
2. Find "ELIMUCORE"
3. Enable notifications
4. Go back to app
```

## â“ Frequently Asked Questions

**Q: Will I lose my data if I switch phones?**
```
A: No! Your data is in the cloud. Log in on new phone, 
   everything syncs automatically.
```

**Q: Can I use the app on tablet?**
```
A: Yes! Works on iPad and Android tablets. 
   Actually better than phone (bigger screen).
```

**Q: Do I need to pay for the app?**
```
A: No! It's free. Included with your ELIMUCORE account.
   No in-app purchases.
```

**Q: What if I forget my password?**
```
A: Use "Forgot Password" on login screen.
   Or ask your school administrator to reset it.
```

**Q: Can I access from multiple devices?**
```
A: Yes! Log in on phone, tablet, and laptop.
   Only one session at a time (new login logs out old).
```

**Q: Is my data private?**
```
A: Yes! All data encrypted in transit and at rest.
   Only you and school admin can see your data.
```

**Q: What if the app crashes?**
```
A: It will auto-recover. Your unsaved data is protected.
   All marks saved before closing are safe.
```

**Q: Can I work offline?**
```
A: Yes! In v1.3.0+, you can enter marks/attendance offline
   and sync when internet is available.
```
